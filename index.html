Privatlivspolitik og Vilkår for TeoriTjek

Sidst opdateret: 01/04/2025

1. Introduktion

TeoriTjek prioriterer dit privatliv højt. Denne politik beskriver vores behandling af data og betingelser for brug af appen.

2. Dataindsamling

  • Ingen personlige data: TeoriTjek kræver hverken login eller registrering.
  • Annoncering: Appodeal SDK leverer annoncer og kan opsamle anonymiseret enhedsdata i overensstemmelse med deres privatlivspolitik.

3. Formål med dataindsamling

  • Gratis app‑brug via annoncefinansiering.
  • Generel forbedring af brugeroplevelsen baseret på anonym statistik.

4. Deling af data

TeoriTjek deler eller opbevarer ikke personlige data. Appodeal håndterer anonymiseret data i henhold til deres privatlivspolitik.

5. Immaterielle rettigheder

Alt indhold i TeoriTjek, inkl. tekst, billeder, logoer og spørgsmål, tilhører TeoriTjek og er beskyttet af ophavsretsloven. Indhold må ikke kopieres, distribueres eller genbruges uden udtrykkeligt samtykke.

6. Ansvarsbegrænsning

TeoriTjek leveres "som det er", uden garanti for fejl eller mangler. Vi er ikke ansvarlige for direkte eller indirekte tab i forbindelse med brugen af appen.

7. Brugerens rettigheder

Brugeren kan til enhver tid afinstallere appen, hvorved dataindsamling ophører.

8. Ændringer

Vi forbeholder os retten til løbende at opdatere denne politik og vilkår. Ændringer offentliggøres her.

9. Kontakt

Kontakt os på: support@teoritjek.dk
